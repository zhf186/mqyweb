# 漫骑游品牌素材库

从品牌手册PDF提取的图片资源，按用途分类整理。

## 目录结构

```
brand_assets/
├── hero/          # 封面/Hero区域图片 (2张)
├── brand/         # 品牌理念图片 (7张)
├── sections/      # 五大板块图标 (5张)
├── business/      # 商业模式图片 (7张)
├── goods/         # 在地好物图片 (34张)
├── gallery/       # 生活馆场景图片 (5张)
├── ebike/         # E-BIKE产品图片 (25张)
├── routes/        # 骑游线路图片 (8张)
├── community/     # 社群活动图片 (9张)
├── org/           # 组织架构图片 (2张)
├── partner/       # 合作伙伴图片 (6张)
└── cities/        # 城市扩张图片 (20张)
```

## 图片用途说明

### hero/ - 首页Hero区域
- `page1_img1.jpeg` - 主视觉大图，适合首页背景
- `page1_img2.jpeg` - 辅助视觉图

### brand/ - 品牌理念
- 用于「关于我们」页面
- 展示品牌核心理念：有深度的线路、有智趣的社群、有品质的好物、有境界的异业、有风景的生活馆

### sections/ - 五大板块
- `page5_img1-5.jpeg` - 五大板块图标（租赁、线路、社群、好物、异业）
- 用于首页板块展示

### business/ - 商业模式
- 展示三大核心战略：产品服务化、服务平台化、收入多元化
- 用于「关于我们」或「商业合作」页面

### goods/ - 在地好物
- 展示骑游沿途的特色好物
- 用于「好物」页面和首页推荐

### gallery/ - 生活馆
- 展示线下生活馆场景
- 用于「生活馆」页面

### ebike/ - E-BIKE产品
- 展示途尔电助力自行车
- 展示智能运营系统（扫码租车、GPS定位、智能锁等）
- 用于「E-BIKE」页面

### routes/ - 骑游线路
- 展示特色线路风光
- 奉化三湖双海、三江古韵、东钱湖、横溪镇等
- 用于「线路」页面

### community/ - 社群活动
- 展示社群活动精彩瞬间
- 用于「社群」页面

### org/ - 组织架构
- 展示六大功能事业部
- 用于「关于我们」页面

### partner/ - 合作伙伴
- 展示政企合作、异业联盟
- 用于「合作」页面

### cities/ - 城市扩张
- 展示已布局城市风光
- 宁波、上海、贵州、北京、成都、深圳、海南、安徽、杭州等
- 用于「发展愿景」板块

## 使用建议

1. **图片优化**：建议使用WebP格式，并生成多尺寸响应式图片
2. **懒加载**：非首屏图片使用懒加载
3. **CDN加速**：上传至阿里云OSS并配置CDN
4. **Alt文本**：为每张图片添加描述性Alt文本，提升SEO和无障碍性
